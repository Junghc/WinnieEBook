# WinnieEBook
> IOS Homework

### App分為三部分:

* 主要角色介紹
* 些許相關電影簡介（採用table controller列表顯示系列電影名稱）
* 關於Winnie The Pooh的Q&A
（文字介紹部分皆使用text view方便滾動瀏覽更多資訊）
